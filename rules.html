Rules

    1. Place the numbers 1-4 once in each row
      and column of the grid. There is no 'box'
      constraint in calcudoku.
    2. Bold-lined regions contain a number and
      mathematical operation. The digits within
      the region must match the given sum.
    3. Numbers are allowed to repeat within
      bold-lined sum regions, as long as doing so
      doesn't break the rule that a number
      cannot repeat within a row or column.
